## 3. Real-World Workflows (Systematic)

### Workflow: The Neural Trend Shift
1.  **Forecaster** scans web for emerging SS25 materials using **Search Grounding**.
2.  **Strategy Agent** uses **Thinking** to determine if the trend aligns with the brand's Heritage DNA.
3.  **Creative Director** generates 3 new shoot concepts based on the validated trend.

### Workflow: Omnichannel Growth Sprint
1.  **Growth Agent** simulates reach for a new campaign concept.
2.  **Storyteller** creates 3 distinct verbal variants (Editorial, Street, Minimal).
3.  **Guardian Agent** validates all variants against the Style Guide.
4.  **n8n Link** pushes the validated mix to global scheduling buffers.
