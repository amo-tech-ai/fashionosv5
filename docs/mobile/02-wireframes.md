# FashionOS Mobile Wireframes

## 1. Dashboard (Mobile)
- **Top Bar**: Brand Logo (Left), Search Icon (Right).
- **Main Canvas**: 
  - Welcome Banner (Stacked vertically).
  - DNA Index (Simplified circle with score).
  - Critical Path (Vertical list with large checkboxes).
- **Bottom Navigation**: Home, Calendar, Shoots, Concierge, Profile.

## 2. Intelligence Bottom Sheet
- **Collapsed State**: Handle indicator, Title, "Neural Sync" pulse. (Visible height: 80px).
- **Expanded State**: Full content, internal custom scrollbar, "Close" button.

## 3. Product Catalog
- **Image**: Full-width header.
- **Stats**: Vertical stack of metrics.
- **Action**: "Brief Shoot" (Fixed bottom primary button).
